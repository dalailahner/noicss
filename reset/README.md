  A modern reset / normalize by noicss.
</p>

## Installation:

```Shell
npm install noicss.reset
```

## Usage:

import it into your CSS file:

```CSS
@layer reset, main;

@import url("noicss.reset") layer(reset);

/* YOUR OWN CSS: */
@layer main {

}
```
