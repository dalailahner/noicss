<h1 align="center">
  noicss.colors
</h1>
<p align="center">
  Color palette as custom CSS properties, that you can customize in vanilla CSS.
</p>

## Installation:

```Shell
npm install noicss.colors
```

## Available colors:

there are 3 categories of colors, each with 5 variations to choose from:

- accentColor-1
- accentColor-2
- accentColor-3
- accentColor-4
- accentColor-5

- bgColor-1
- bgColor-2
- bgColor-3
- bgColor-4
